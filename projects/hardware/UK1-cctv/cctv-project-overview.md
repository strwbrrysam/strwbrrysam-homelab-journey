# CCTV installation project at UK1 site

This little folder documents the process I went through to install CCTV cameras at my UK1 site. 

The process involved:
- Drilling holes through walls
- Lifting paving slabs and digging small trenches
- Laying conduit both rigid and flexible
- Pulling ethernet cable
- Terminating ethernet cable
- Testing etherner cable
- Installing junction boxes
- Installing Annke outdoor dome cameras
- Configuring the Annke system 
- Ensure connectivity over VPN for remote viewing
